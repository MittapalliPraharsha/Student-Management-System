# Student Management System
## Description
A File-Based Persistent Student Management System developed using Core Java.
This application stores student records permanently in a text file and retrieves them whenever the program restarts.
## Features
- Add Student
- View Students
- Search Student by ID
- File-Based Data Persistence
## Technologies Used
- Java
- File Handling
- FileWriter
- BufferedReader
- ArrayList
- OOP Concepts
## Files Included
- Student.java
- FileManager.java
- StudentApp.java
- students.txt
- output.png
## How to Run
Compile:
javac Student.java FileManager.java StudentApp.java
Run:
java StudentApp
## Sample Data Format
1,Harsha,20
2,Praharsha,21

- Data Persistence
